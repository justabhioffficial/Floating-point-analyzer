About The Project

The Floating Point Error Analyzer is a production-grade demonstration of numerical engineering problems, built with a Flask Python backend and Vanilla JavaScript frontend.

Analysis Engine

Built with NumPy, providing robust safety when executing mathematical equations into numerical operations.

Dynamic Visualizations

Utilizes Chart.js to map rounding and truncation error disparities interactively across arbitrary precision domains.

Diagnostic Explanations

Generates descriptive diagnostic explanations indicating why certain numerical phenomena happen under certain operations.

Reporting Engine

Exports metric histories directly to downloadable PDF archives utilizing the robust ReportLab package.
